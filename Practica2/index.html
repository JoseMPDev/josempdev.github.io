Referencia de 102 Etiquetas HTML

Etiqueta Función Ejemplo
<!DOCTYPE> Define el tipo de documento HTML <!DOCTYPE html>
<html> Define la raíz de un documento HTML <html lang="es">...</html>
<head> Contiene metadatos/información para el documento <head>...</head>
<title> Define el título del documento <title>Título de la página</title>
<body> Define el cuerpo del documento <body>...</body>
<h1> - <h6> Define encabezados HTML <h1>Encabezado principal</h1>
<p> Define un párrafo <p>Este es un párrafo.</p>
<br> Inserta un salto de línea Línea 1<br>Línea 2
<hr> Define un cambio temático en el contenido <hr>
<!--...--> Define un comentario <!-- Este es un comentario -->
<a> Define un hipervínculo <a href="https://ejemplo.com">Enlace</a>
<img> Define una imagen <img src="imagen.jpg" alt="Descripción">
<div> Define una sección en un documento <div>Contenido</div>
<span> Define una sección en línea en un documento <span>texto con estilo</span>
<ul> Define una lista desordenada <ul><li>Item</li></ul>
<ol> Define una lista ordenada <ol><li>Item 1</li></ol>
<li> Define un item de lista <li>Item de lista</li>
<table> Define una tabla <table>...</table>
<tr> Define una fila en una tabla <tr><td>Celda</td></tr>
<th> Define una celda de encabezado en una tabla <th>Encabezado</th>
<td> Define una celda en una tabla <td>Datos</td>
<form> Define un formulario HTML <form action="/enviar">...</form>
<input> Define un control de entrada <input type="text" name="nombre">
<textarea> Define un control de entrada multilinea <textarea rows="4" cols="50"></textarea>
<button> Define un botón cliqueable <button type="button">Clic aquí</button>
<select> Define una lista desplegable <select><option>Opción</option></select>
<option> Define una opción en una lista desplegable <option value="1">Opción 1</option>
<label> Define una etiqueta para un elemento de entrada <label for="nombre">Nombre:</label>
<fieldset> Agrupa elementos relacionados en un formulario <fieldset><legend>Datos</legend>...</fieldset>
<legend> Define un título para un <fieldset> <legend>Información personal</legend>
<iframe> Define un marco inline <iframe src="pagina.html"></iframe>
<audio> Define contenido de sonido <audio controls><source src="audio.mp3"></audio>
<video> Define un video o película <video controls><source src="video.mp4"></video>
<source> Define recursos para elementos multimedia <source src="video.mp4" type="video/mp4">
<track> Define texto de pista para elementos multimedia <track src="subtitulos.vtt" kind="subtitles">
<canvas> Define gráficos dibujados con script <canvas id="miCanvas" width="200" height="100"></canvas>
<meta> Define metadatos sobre un documento HTML <meta name="description" content="Descripción">
<style> Define información de estilo para un documento <style>body {color: red;}</style>
<link> Define la relación entre un documento y un recurso externo <link rel="stylesheet" href="estilo.css">
<script> Define un script del lado del cliente <script src="script.js"></script>
<noscript> Define un contenido alternativo para usuarios sin soporte de scripts <noscript>JavaScript no habilitado</noscript>
<section> Define una sección en un documento <section><h2>Título</h2>...</section>
<article> Define un artículo independiente <article><h2>Noticia</h2>...</article>
<aside> Define contenido aparte del contenido principal <aside>Nota lateral</aside>
<header> Define un encabezado para un documento o sección <header><h1>Logo</h1></header>
<footer> Define un pie de página para un documento o sección <footer>Copyright 2023</footer>
